## 技术信息
- 原始程序由 Python 3.14 编写，使用 PyInstaller 打包
- 使用 Playwright 做浏览器自动化抓取直播弹幕
- 使用 Tkinter 做 GUI 界面
- 通过 IRC 协议（localhost:6667）桥接弹幕到游戏
- 支持平台：抖音、B站、小红书
